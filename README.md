# 아주대학교 SW 캡스톤디자인
## Smart Biclcye Assistant(S-BA)


### 프로젝트 개요
본 프로젝트는 기존 자전거 어플리케이션들의 장점을 모아 자전거에 특화된 통합 어플리케이션을 제작하고자 하였다. 기존 앱들에서 경로 안내, 주행 기록 관리, 주행 중 지도와 주행 정보 전달 기능을 가져왔으며 기존에 없었던 자전거 유지보수 기록 관리 기능, 후방 차량 감지 기능, 단체 라이딩 추천기능을 추가하였다.
